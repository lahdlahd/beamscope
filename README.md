# beamscope

What is BeamScope?
BeamScope visualizes the health of the Beam gaming ecosystem. It tracks daily active wallets, $BEAM transaction volume, NFT trading activity, and player growth across all Beam games — in one clean dashboard.
Features

Ecosystem Overview — Total players, volume, transactions, and active games at a glance
Growth Charts — Daily active players and transaction trends over time
Trending Games — Ranked by daily active players with change indicators
$BEAM Flow — Transaction breakdown by type (gaming, NFT, DeFi)

Tech Stack

React
Recharts (data visualization)
Beam on-chain data (planned)

Run Locally
bashnpm install
npm run dev
